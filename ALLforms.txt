Afsprakenblok

Gebruik altijd het standaard wijzigformulier: ADD / REPLACE / DELETE / CREATE.
Antwoord steeds door exact één formulier in te vullen (labels onveranderd), met:

Markers verplicht bij code: gebruik mijn vaste marker-structuur (# [SECTION: …] / # [END: SECTION: …]).

Bij Python: markers rond imports, top-level functies, classes, en main-guard.

Markers komen alleen boven/onder blokken; niet binnen bodies.

Logging verplicht:

Bij nieuwe scripts: voeg basis logging toe (module-logger, start/stop meldingen bij main/entry-points, korte debug in kernfuncties).

Bij wijzigingen: behoud bestaande logger; voeg waar nuttig 2-3 duidelijke logregels toe (geen overkill).

Contextregels = 3 (tenzij anders nodig).

REPLACE/DELETE: Marker-van en Marker-tot moeten exact matchen met het doelbestand.

Voorstel-blok bevat het volledige blok (incl. markers).

CREATE: vul ook JSON-registratie, commit-boodschap en “Openen-in”.

Defaults:

JSON registreren = ja (in .projassist.json, groep scripts).

Git = ja (commit en push).

Encoding = utf-8.

Bestaat-al (CREATE) = mislukken tenzij ik anders vraag.

Commit-boodschap: kort, beschrijvend.

Antwoord nooit met code buiten het formulier; alleen het formulier zelf.
